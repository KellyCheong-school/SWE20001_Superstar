Steps to launch the website:
1. Download/Clone this repository (main branch)
2. Install MAMP application for Windows
3. Paste the "Superstar" folder in MAMP directory (i.e., C:\MAMP\htdocs)
4. Launch MAMP application (make sure Apache Server & MySQL Server is turned on)
5. On your browser, navigate to the database (i.e., http://localhost/phpMyAdmin/)
6. Create a "New" database with the name "flightbooking"
7. Press "Inport" > "Choose File"
8. Choose the .sql file from the "Database" folder
9. Then, press "Go"
10. To launch the website on your browser, paste "http://localhost/Superstar"
